# Sistema Bancário Simples em Python

Este projeto é um sistema bancário simples feito em Python, criado para fins de **estudo e prática de programação**.

## Funcionalidades

- Saque de valores
- Depósito de valores
- Consulta de saldo
- Validação de valores inválidos
- Menu interativo no terminal

## Como funciona

O sistema inicia com um saldo padrão e permite que o usuário escolha opções através de um menu no terminal.

As principais funções do sistema são:
- `sacar(saldo)`
- `depositar(saldo)`
- `sistema()`

Cada operação retorna o saldo atualizado para garantir o controle correto do valor.
